# REFERENCE DATA -- Keywords, LSI, Glossario e Tabelas

## 1. POWER KEYWORDS

### 1.1 Clusters Informacionais

**CLUSTER 1 -- Autoridade e Controle Semantico**
Uso: sinalizar dominio do assunto em H1, H2, meta descriptions.
Guia completo, Guia definitivo, Entenda de vez, O que realmente importa, Tudo o que voce precisa saber, Explicacao completa, Analise aprofundada, Visao especializada, Analise critica, Abordagem tecnica

**CLUSTER 2 -- Educacao e Processo**
Uso: tutoriais, guias, conteudo educacional.
Como aprender, Desenvolvimento real, Evolucao pratica, Aprendizado consciente, Formacao solida, Pratica estruturada, Metodo de aprendizado, Construcao gradual, Desenvolvimento continuo, Passo a passo

**CLUSTER 3 -- Etica e Confianca**
Uso: conteudo sobre boas praticas, conformidade.
Uso etico, Uso responsavel, Uso consciente, Dentro das regras, Sem penalizacoes, Pratica segura, Criterios claros, Avaliacao justa, Integridade, Conformidade

**CLUSTER 4 -- Diferenciacao e Posicionamento**
Uso: evitar conteudo generico, posicionar como referencia.
Diferenca entre, O que muda na pratica, Quando vale a pena, Onde esta o erro, O que poucos explicam, Como funciona de verdade, O que faz diferenca, Abordagem correta, Uso adequado, O que ninguem te conta

**CLUSTER 5 -- Intencao Investigativa e Comparativa**
Uso: H2 comparativos, snippets, PAA.
Vale a pena, Quando usar, Quando nao usar, Pros e contras, Vantagens e limitacoes, Beneficios reais, Riscos envolvidos, Limites do uso, Comparacao pratica, Analise objetiva

**CLUSTER 6 -- Metodo e Estrutura**
Uso: conteudo tecnico, processos.
Passo a passo, Criterios de avaliacao, Metodo correto, Processo estruturado, Avaliacao criterial, Estrutura ideal, Modelo recomendado, Abordagem orientada, Padroes, Sequencia logica

**CLUSTER 7 -- Resultado e Transformacao**
Uso: com cuidado, sem promessas irreais.
Resultados consistentes, Evolucao mensuravel, Melhoria continua, Desenvolvimento real, Progresso mensuravel, Ganho de eficiencia, Aprimoramento, Otimizacao, Avancos concretos, Impacto positivo

**CLUSTER 8 -- Problemas e Alertas**
Uso: capturar atencao sem alarmismo.
Erros comuns, Principais falhas, O que evitar, Onde muitos erram, Riscos reais, Problemas frequentes, Limitacoes praticas, Uso incorreto, Dependencia excessiva, Falhas tipicas

**CLUSTER 9 -- Tecnologia e Modernidade**
Uso: conteudo tecnico, sem hype.
Tecnologia aplicada, Ferramenta de apoio, Feedback automatizado, Avaliacao assistida, Tecnologia especializada, Solucao digital, Processo automatizado, Plataforma dedicada, Sistema integrado, Recurso tecnologico

**CLUSTER 10 -- Clareza e Transparencia**
Uso: GEO, citabilidade por LLMs.
De forma clara, Sem atalhos, Sem promessas irreais, O que e e o que nao e, Limites e possibilidades, Na pratica, Em contexto real, Com exemplos, De maneira objetiva, Explicado passo a passo

### 1.2 Keywords Comerciais

**Decisao e Compra:**
Vale a pena, Melhor opcao, Melhor escolha, Solucao ideal, Alternativa confiavel, Opcao recomendada, Escolha certa, Melhor custo beneficio, Opcao mais segura, Solucao eficiente

**Comparacao e Avaliacao:**
Comparativo, Comparacao completa, Analise comparativa, Diferenca entre, Comparado a, Avaliacao completa, Teste pratico, Analise detalhada, Avaliacao real, Comparacao honesta

**Valor e Retorno:**
Custo beneficio, Investimento inteligente, Retorno esperado, Ganho real, Economia de tempo, Reducao de esforco, Eficiencia comprovada, Valor agregado, Impacto pratico, Beneficio direto

**Confianca e Prova:**
Funciona mesmo, Resultados reais, Uso comprovado, Solucao confiavel, Metodo validado, Experiencia pratica, Casos reais, Testado na pratica, Avaliado por usuarios, Comprovacao pratica

**Adocao e Uso:**
Como escolher, Quando usar, Para quem e indicado, Para quem nao e indicado, Como aplicar, Implementacao pratica, Uso recomendado, Escolher corretamente, Tomar decisao, Comecar agora

### 1.3 Regras de Uso

- Informacionais: usar em H1, H2, meta descriptions, primeiro paragrafo, FAQ
- Comerciais: usar em landing pages, comparacoes, CTAs, secoes de decisao
- Distribuir naturalmente, nunca acumular no mesmo paragrafo
- Combinar com entidades do topico para reforcar relevancia semantica
- Maximo 2 power keywords por secao H2

**Contextuais:** sistema de saude, plano de saude, atendimento, emergencia, internacao

**Secundarios:** tributacao, custos, diversificacao, carteira, aporte, resgate, vencimento

**Secundarios:** desconto, promocao, parcelamento, cashback, cupom, comparativo, estoque

**Contextuais:** frete gratis, entrega expressa, devolucao, satisfacao, atendimento ao cliente

**Primarios:** edital, vagas, concurso, inscricoes, prova, banca, remuneracao, cargo, cronograma

## 3. GLOSSARIO TECNICO

| Termo | Definicao |
|-------|-----------|
| **Entity Lock-in** | Fixacao da entidade-alvo nos primeiros 100 palavras |
| **LSE** | Latent Semantic Entities. Embeddings de entidades em grafos semanticos |
| **GEO** | Generative Engine Optimization. Otimizacao para citacao por LLMs |
| **E.E.A.T.S.** | Entity-Encoded Article Topical Structure. 7 camadas de analise semantica |
| **MTT** | Meaning-Text Theory. Significado por combinacoes lexicais |
| **COINC** | Metrica de coincidencia semantica: POS_SIM - NEG_SIM |
| **FSW** | Frequency-balanced Semantic Weight. Termos raros = maior peso |
| **SAF** | Semantic Approximation Factor. Qualidade do match |
| **EAV** | Entity-Attribute-Value. Modelo de organizacao de conhecimento |
| **Summarization** | Resumo GEO de 50-60 palavras no inicio do artigo |
| **Topical Authority** | Autoridade por cobertura semantica completa e conectada |
| **Information Gain** | Quanto o conteudo reduz a incerteza do sistema |
| **Semantic Distance** | Distancia conceitual entre secao e entidade central |
| **REL Factor** | Peso da posicao: titulo > H2 > body text |
| **BERT** | Bidirectional Encoder Representations from Transformers |
| **MUM** | Multitask Unified Model. 1000x BERT, 75 idiomas, multimodal |
| **SGE** | Search Generative Experience. Respostas generativas na SERP |
| **RAG** | Retrieval-Augmented Generation. Recuperacao + geracao por LLM |
| **Synset** | Grupo de sinonimos representando um conceito unico (WordNet) |
| **Semfield** | Campo semantico amplo que agrupa multiplos synsets |
| **Topical Map** | Ontologia editorial hierarquica do site |
| **Silo** | Estrutura de conteudo organizada em clusters tematicos com hub central |
| **Child Page** | Pagina especifica que pertence a um silo e aponta para o hub |
| **Hub Page** | Pagina central do silo que distribui autoridade para child pages |

### 4.2 Lexical Functions

| LF | Relacao | Exemplo (costura) |
|----|---------|-------------------|
| Syn0 | Mesmo significado | costura = trabalho de agulha |
| Syn1 | Similar | costura = confeccao |
| Syn2 | Relacionado | costura = artesanato textil |
| Oper | Verbo associado | costura = costurar, alinhavar |
| Vo | Verbalizacao | costura = costurar |
| Ao | Adjetivacao | costura = costureira, artesanal |

### 4.3 Relacoes WordNet

| Relacao | Direcao | Exemplo | Uso SEO |
|---------|---------|---------|---------|
| Synonymy | bidirecional | costura = confeccao | Variacao |
| Antonymy | bidirecional | costurar x descosturar | Contraste |
| Hypernymy | para cima | costura -> artesanato | Contexto |
| Hyponymy | para baixo | artesanato -> costura | Especificidade |
| Holonymy | todo | vestido -> manga | Componentes |
| Meronymy | parte | manga -> vestido | Detalhamento |
